# CoutureLaFleur

A premium Rwandan luxury fashion brand website blending French couture with African aesthetics.

## Tech Stack
- **Frontend:** React, TailwindCSS, Lucide Icons
- **Backend:** Node.js, Express, MongoDB
- **Payments:** Stripe & Mobile Money Integration

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Seed the database (Mock Data):
   ```bash
   npm run seed
   ```

3. Start the server:
   ```bash
   npm start
   ```
